# Install

	p keepass

# NOTE: keepass installation directory = [	/usr/share/keepass	]


# Import from Lastpass

	Steps:

	1: Export passwords from Lastpass in "csv" format

	2: Rename the lastpass file to *.csv

	3: Open the file in Libreoffice calc

	4: Edit the file to the following format:
		"Name","User Name","Password","URL","Comments"

	5: Select everything, right-click and select 'Format Cells'. 
	   In the dialog, choose Text as category. Click [OK].

	6: Go to 'File' -> 'Save As...', 
	   choose a location and make sure that the 'Edit Filter Settings' 
	   box on the bottom left is enabled. Click the 'Save' button.

	7: Choose 'Unicode (UTF-8)' as character set.
	   Set field separator as a comma. And Text separator as a dot
	   Make sure that the "Fixed Column Width box" is not checked. Click [OK].

	8: Import into Keepass using the Generic csv Importer
